# Sales Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard analyzing sales performance, profitability, and customer behavior.

## Dataset
- Retail / sales dataset
- Includes revenue, profit, quantity, and customer segments

## Tools Used
- Power BI
- Power Query (data cleaning)
- DAX (calculated measures)

## Key Insights
- Identified top-performing products and regions
- Detected periods of declining revenue
- Analyzed profit margins across categories
- Highlighted customer purchasing patterns

## Features
- Interactive filters (date, region, product)
- Dynamic KPIs (revenue, profit, growth)
- Drill-down capability for deeper analysis
